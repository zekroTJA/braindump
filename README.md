# Preamble

In 2022, I set a goal to document everything around what I learn, what I work with or what I am discovering in a public repository of knowledge. Back then, I used obsidian and a plugin to push the database to a [Git Repository on GitHub](https://github.com/zekroTJA/resource-vault/). But sadly, I stopped using [Obisdian](https://obsidian.md/) and so I lost the motivation to document all my progress.

Therefore, in 2023, I decided to re-create this repository on Notion. A platform which I am now using since a couple of years for my personal notes and TODOs. So I feel quite comfortable to document all the stuff here. The whole workspace is then exported automatically on a daily basis to a [Git Repository on GitHub](https://github.com/zekroTJA/braindump/) using [this fancy tool](https://github.com/darobin/notion-backup). 

Primarily, this is purely for documenting stuff I might forget but really need to find later. So don’t expect high quality documentation here. Also, this does not represent *everything* I have in my mind — at least yet in the current state.

# Contribution

Feel free to create an issue if you have any idea on how to improve these docs or if you want to add missing information. **Pull Requests will be ignored** because back-merging into Notion from these export is quite difficult. Simply express your change-suggestions in an issue and I might add your contributions to the docs (with credits to you if desired, of course).
