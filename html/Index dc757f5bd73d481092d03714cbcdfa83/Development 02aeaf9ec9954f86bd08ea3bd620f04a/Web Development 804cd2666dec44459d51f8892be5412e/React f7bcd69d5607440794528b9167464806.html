styled-components

A very great library that let’s you create components with CSS/SCSS style defined directly in the JavaScript code. This way, you can reference and react to variables in your app. Also, this allows to dynamically theme your app using the ThemeProvider.

For a better developer experience with Visual Studio Code, I suggest installing the vscode-styled-components extension. It enables intellisense and code highlighting in these style tags.

Example

const Button = styled.button`
	border-radius: 3px;
  background-color: ${(p) => p.theme.accent};
`;

Links

  • Project: https://github.com/styled-components/styled-components
  • Docs: https://styled-components.com/docs

zustand

Absolutely dead simple React State library which is a good alternative to React Redux for small to medium sized apps.

Example

// Store.ts

export interface Store {
  counter: number;
  setCounter: (v: number) => void;
  increaseCounter: () => void;
}

export const useStore = create<Store>((set, get) => {
  counter: 0,
  setCounter: (counter) => set({ counter }),
  increaseCounter: () => set({ counter: get().counter + 1 }),
});

// Components.ts

export const Component: React.FC = () => {
  const [counter, increaseCounter] = useStore((s) => [
    s.counter, 
    s.increaseCounter,
  ]);

  return <button onClick={() => increaseCounter()}>
		{counter}
  </button>;
}

react-uid

A simple solution for generating UIDs for the key property in React element lists.

Example

export const Component: React.FC = () => <div>
	{[1, 2, 3].map((v) => <p key={uid(v)}>{v}</p>)}
</div>;

  • Project: https://github.com/thearnica/react-uid
